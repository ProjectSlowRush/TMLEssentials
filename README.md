# TMLEssentials

Essential tools for modding Terraria

At the moment, the only tool it has is an Equip Texture previewer, which allows you to preview relevant animations of your armor sets before having to build, compile, and play your mods. Also included are a couple basic playback controls for animations.

[Click here to download this application. Download both the `.pck` and your respective operating system's executable file.](https://github.com/EtherealCrusaders/TMLEssentials/releases/latest)
